# Sushi Counter — Brand Identity Guide

> *Track. Compete. Dominate.* 🍣🏆

---

## 1. Brand Overview

| Attribute | Detail |
|---|---|
| **Name** | Sushi Counter |
| **Tagline** | Track. Compete. Dominate. |
| **What it is** | Gamified sushi-eating leaderboard for sushi nights with friends |
| **Vibe** | Fun, competitive, playful, social |
| **Tone** | Energetic, cheeky, lighthearted — like a group chat that turned into a competition |
| **Platform** | Mobile-first web app |

**Who it's for:**
- Friend groups who go out for sushi together
- People who love tracking stats and competing
- Sushi enthusiasts who want bragging rights

## 2. Color Palette

```
▮▮▮▮▮ #FF6B6B  Salmon Coral — Primary brand color (salmon/tuna vibes)
▮▮▮▮▮ #7BC67E  Matcha Green — Secondary (wasabi & gari vibes)
▮▮▮▮▮ #FFD93D  Tamago Yellow — Accent (egg, fun, highlight)
▮▮▮▮▮ #2D2D2D  Charcoal — Text, high contrast
▮▮▮▮▮ #FFF8F0  Rice White — Backgrounds, cards
▮▮▮▮▮ #FF8C42  Salmon Orange — Warm accent, CTAs
```

### Usage Rules
- **Salmon Coral** = primary for buttons, app icon, headers
- **Matcha Green** = secondary buttons, success badges, nature element
- **Tamago Yellow** = highlights, special badges, confetti/sparkle effects
- **Rice White** = main background color
- **Charcoal** = all body text

---

## 3. Typography

### Primary: **Poppins** (sans-serif, rounded)
- Playful but clean — perfect for app UI
- Use for headings, buttons, leaderboard names

### Secondary: **Inter** (sans-serif)
- Use for body text, stats, small details
- More legible at small sizes

### Display: **Fredoka One** (rounded display)
- Use for BIG NUMBERS on the counter
- Fun, bold, chunky

### Hierarchy
```
App Title — Fredoka One, 32pt
Headline — Poppins Bold, 24/20/18pt
Body — Poppins Regular, 16/14pt
Counter Number — Fredoka One, 72/64/48pt
Small — Inter Medium, 12pt (footnotes, timestamps)
```

## 6. UI Design System

### Components

| Component | Style |
|---|---|
| **Buttons** | Rounded (16px radius), coral primary, green secondary |
| **Cards** | White with subtle shadow, rounded (12px radius) |
| **Inputs** | Underlined or soft border, rounded |
| **Badges** | Small pill shapes, coral or gold for #1 |
| **Progress bars** | Coral fill on white track |
| **Avatars** | 40px circle, with rank badge overlay |

### Leaderboard Specific

- **#1 spot:** Gold crown + larger avatar + sparkle effect
- **#2 spot:** Silver, slightly smaller
- **#3 spot:** Bronze
- **Your row:** Highlighted coral background

### Counter Screen
- **Big number:** Fredoka One, 72pt, coral
- **+1 button:** Large pill, green ("Eat another!" 🍣)
- **Recent entries:** Horizontal scroll of sushi emojis
- **Session timer:** Small, top-right

## 7. Tone & Copywriting

| Do | Don't |
|---|---|
| Short, punchy, energetic | Corporate or formal |
| Friendly trash talk | Actually mean |
| Sushi puns (yes, *all* of them) | Taking yourself seriously |
| "You ate 42 pieces? Legend." | "Your consumption has been recorded." |

### Sample Copy

| **Friend passes you** | "Uh oh. @Marko just took your spot. 😤" |
| **Counter button** | "🍣 +1" / "🔥 Keep going!" |
| **End of night** | "Sushi night over! Final standings 👇" |
| **Weekly recap** | "This week you ate 28 pieces. Respect. 🫡" |

## 8. Mobile App Screens

### Screen 1: **Leaderboard**
- Friends ranked by total sushi count
- Current session toggle (tonight / all time / this week)
- Pull to refresh
- Your rank highlighted

### Screen 2: **Counter**
- Big sushi count number in center
- +1 button with satisfying animation
- Minus button (oops, I miscounted)
- Recent items strip at bottom

### Screen 3: **Profile / Stats**
- Total all-time count
- Best single session
- Average per session
- Streak (consecutive sushi nights)
- Badges earned

### Screen 4: **Sushi Night**
- Create / join a sushi night
- Set a goal (e.g., "50 pieces between us")
- Live count per person
- End night → final leaderboard → share card
